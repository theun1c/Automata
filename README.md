
# 🍬 Информационная система для сети вендинговых аппаратов

Десктопное приложение для автоматизации управления, мониторинга и аналитики сети вендинговых автоматов. Разработано на C# с использованием Avalonia UI и PostgreSQL для хранения данных.

## 📋 О проекте

Система предназначена для франчайзеров и операторов вендингового бизнеса. Она объединяет в себе инструменты контроля запасов, технического состояния аппаратов и аналитики продаж, что позволяет:

- 🚀 Снизить операционные затраты за счёт автоматизации учёта и прогнозирования
- 📈 Повысить эффективность работы каждого аппарата
- 😊 Улучшить пользовательский опыт благодаря своевременному пополнению товаров и предотвращению простоев

### 🎯 Цели и задачи системы

- **Управление запасами**: автоматизированный мониторинг наличия товаров в аппаратах, прогнозирование спроса и оптимизация процессов пополнения запасов
- **Техническое обслуживание**: фиксация состояния аппаратов, уведомления о необходимости технического обслуживания или ремонтах
- **Аналитика и отчетность**: сбор данных о продажах, популярных товарах и времени пиковых продаж для проведения анализа и принятия обоснованных бизнес-решений

### 🏗️ Компоненты системы

| Компонент | Описание |
|-----------|----------|
| 🖥️ **Интерфейс управления** | Приложение для франчайзера, где можно видеть статус всех вендинговых машин, управлять запасами и получать уведомления |
| 📡 **Мониторинг в реальном времени** | Использование датчиков для отслеживания состояния аппаратов и товаров (наличие определенных товаров, наличие денег в аппарате) |
| 🗄️ **База данных** | Хранение информации о товарах, продажах, операциях и пользователях системы |
| 📊 **Аналитические инструменты** | Графики и отчеты для анализа производительности аппаратов и предпочтений клиентов |

## ✨ Функциональные возможности

### 📦 Управление запасами
- Автоматический мониторинг остатков товаров
- Прогнозирование спроса на основе истории продаж
- Оптимизация маршрутов и графиков пополнения

### 🔧 Техническое обслуживание
- Фиксация состояния каждого аппарата (онлайн/офлайн, ошибки)
- Уведомления о необходимости ремонта или профилактики
- История обслуживания и ремонтов

### 📊 Аналитика и отчётность
- Сбор данных о продажах в разрезе товаров, времени, локаций
- Отчёты по доходности, оборачиваемости, популярности позиций
- Графики пиковых нагрузок для планирования ассортимента

## 🛠️ Технологический стек

- **Язык программирования**: C# 
- **UI-фреймворк**: Avalonia UI 
- **База данных**: PostgreSQL 
- **ORM**: Entity Framework Core 
- **Архитектура**: MVVM / Clean Architecture / Feature-First (need a choice)
