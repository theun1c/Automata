# Tomorrow Quick Prompts

Короткие промпты для отдельных задач (copy-paste).

## 1) Pixel-perfect навигация
```text
Сфокусируйся только на навигации desktop/web (sidebar + top bars).
Сверь с assets и доведи до пиксель-перфекта: отступы, размеры, цвета, состояния hover/active, иконки.
Работай в UI-only. В конце покажи diff по файлам.
```

## 2) Только модалки ТА
```text
Проверь модальные окна ТА (create/edit/delete/unbind): визуал, выравнивание, поведение открытия/закрытия.
Никаких изменений бизнес-логики. Только UI/UX.
```

## 3) Только модалки компаний
```text
Доведи модальные окна компаний (add/edit/delete) до соответствия макету.
Сохрани текущую механику ShowDialog(owner).
```

## 4) Аудит архитектуры Desktop
```text
Сделай аудит архитектуры Automata.Desktop и предложи целевую модульную структуру
(Shell, Modules/Auth, Modules/Dashboard, Modules/Monitor, Modules/Admin/*, Common).
Нужен план миграции по шагам с минимальным риском поломок.
```

## 5) Рефакторинг одного модуля
```text
Перенеси только модуль Admin/Machines в новую структуру папок и namespace.
После переноса проверь x:Class, namespaces, ViewLocator, и выдай список измененных файлов.
```

## 6) Web inside Desktop (подготовка)
```text
Подготовь технический план встраивания Razor Monitor в Desktop через WebView + локальный запуск Automata.Web.
Пока без полной реализации, нужен каркас и список рисков.
```

## 7) Ревью текущего состояния
```text
Сделай code review текущего UI: сначала критичные проблемы, потом средние, потом мелкие.
Ссылайся на конкретные файлы и давай практичные фиксы.
```

## 8) Коммиты в стиле git flow
```text
Сгруппируй текущие изменения в атомарные коммиты.
Используй сообщения в формате feat/fix/refactor/docs/chore на английском.
Покажи, какие файлы входят в каждый коммит.
```

## 9) Финальный handoff
```text
Обнови docs/NEXT_SESSION.md и добавь новый docs/SESSION_REPORT_YYYY-MM-DD.md:
что сделано, что осталось, риски, с чего начать следующую сессию.
```

## 10) Safe-mode
```text
Не меняй код. Сначала прочитай контекстные docs и выдай пошаговый план на эту сессию
с оценкой рисков и очередностью задач.
```
