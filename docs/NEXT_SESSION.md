## 2) Текущие технические ограничения

- В среде разработки Codex `dotnet build` падает с:
  - `Build FAILED`
  - `0 Warning(s)`
  - `0 Error(s)`
- При этом:
  - XAML-файлы валидны (`xmllint`),
  - приложение у вас локально запускается.
- Вероятная причина: проблема design-time/build окружения (SDK/workload resolver), а не явная ошибка XAML/C#.
