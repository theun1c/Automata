# AGENTS.md

Этот репозиторий реализует информационную систему для сети вендинговых аппаратов.

## Главный документ
Перед изменениями обязательно прочитай `docs/CODEX_SPEC.md`.

## Главный принцип
Текущая версия проекта должна быть **максимально простой**.

- Не добавляй лишние сущности и таблицы.
- Не закладывай расширения «на будущее».
- Если задачу можно решить проще — решай проще.

## База данных
БД должна быть в схеме `automata`.

В текущей версии допускаются только таблицы:
- `vending_machines`
- `products`
- `sales`
- `users`
- `maintenance_records`
- `machine_statuses`
- `machine_models`
- `roles`

Никаких других таблиц добавлять нельзя, пока пользователь не попросит отдельно.

## Авторизация
- Подтверждение email не нужно.
- Таблицы для кодов подтверждения не нужны.
- Пароли хранить только в хеше.

## Правила
- UI должен быть на русском языке.
- Не ломай стек без необходимости.
- Держи модель данных в простой 3НФ.
- Если меняешь БД, обновляй и SQL-схему, и seed.
